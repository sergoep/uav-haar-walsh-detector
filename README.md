# Calibrated Haar--Walsh Structural Features for UAV Signal Monitoring

This repository contains reproducible Python code for the paper:

**Calibrated Haar--Walsh Structural Features for Lightweight Interference Detection in Short UAV Signal Windows**

The code implements a lightweight Haar--Walsh structural monitoring pipeline for short signal windows. It generates synthetic interference scenarios, computes baseline and proposed scores, applies robust median/MAD normalization, performs split-conformal calibration, and produces benchmark tables and figures.

## Contents

* `run_synthetic_uav_haar_walsh.py` — main synthetic benchmark script.
* `run_real_gnss_haar_walsh.py` — real-data validation runner for public GNSS/RF I/Q datasets.
* `requirements.txt` — Python dependencies.
* `LICENSE` — MIT license.
* `README.md` — repository documentation.

The synthetic script is intended to generate:

* `benchmark_results.tex`
* `conformal_guarantee_check.tex`
* `theoretical_FH_bound_check.tex`
* `timing_benchmark.tex`
* `roc_curves.pdf`
* `scale_energy_profiles.pdf`
* `heatmap_auc_fast_modulation.pdf`

The real-data runner is intended to generate:

* `real_data_dataset_summary.tex`
* `real_data_results.tex`
* `results_real/figures/realdata_roc_curves.pdf`

## Reproducibility

Python version: 3.10 or later.

Install dependencies:

`pip install -r requirements.txt`

Run the synthetic benchmark:

`python run_synthetic_uav_haar_walsh.py`

Run the public GNSS/RF real-data validation protocol after downloading the required public datasets:

`python run_real_gnss_haar_walsh.py`

## Scope and safe use

This repository is limited to offline defensive detector auditing, synthetic benchmark generation, and reproducible analysis of public GNSS/RF I/Q datasets.

It does not provide RF emission parameters, jamming procedures, spoofing procedures, UAV control-channel manipulation, real-time disruption methods, SDR attack scripts, or operational deployment instructions.

## Data

Synthetic data are generated by the Python scripts.

Large public raw I/Q datasets are not redistributed in this repository. Users should download public GNSS/RF datasets from their original sources and use the provided real-data runner or manifest-based processing workflow.
